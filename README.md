
# Thai Flames Network

A high-trust matchmaking ecosystem for sustainable partnerships, featuring AI-assisted vetting and cultural mentorship.

## Tech Stack
- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **AI**: [Google Gemini API](https://ai.google.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Database**: [Vercel Postgres](https://vercel.com/docs/storage/vercel-postgres)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)

## Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/bking86/ThaiFlamesNetwork26.git
   cd ThaiFlamesNetwork26
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Environment Variables**:
   Create a `.env.local` file and add your credentials (do not commit this file):
   ```env
   API_KEY=gen-lang-client-0478239032
   POSTGRES_URL=your_vercel_postgres_url
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```

## Project Structure
- `/app`: Routing and layout logic.
- `/components`: Reusable UI modules (Profile cards, Academy modules).
- `/lib`: Utility functions and API clients (Gemini, DB).
- `/types`: Shared TypeScript interfaces.
